# Proyecto Web Scaffold

Este es el scaffolding inicial del proyecto.

## Estructura de Carpetas

- `index.html`: Página principal (Landing Page).
- `css/`: Estilos globales y variables de diseño.
- `js/`: Lógica principal de la aplicación.
- `screen/`: Sección específica solicitada, con su propio HTML, CSS y JS.
- `assets/`: Carpeta para imágenes y recursos estáticos.

## Cómo empezar

1. Abre `index.html` en tu navegador.
2. Navega a `screen/index.html` para ver la sección interna.
3. Edita `css/style.css` para cambiar los colores globales (variables en `:root`).

## Diseño

El proyecto utiliza un diseño moderno "Dark Mode" con tipografía 'Outfit' de Google Fonts y colores violeta/rosa vibrantes. Todo es responsive.
